# Recuperação - Desenvolvimento de Algoritmos

Repositório destinado às atividades de recuperação da UC Desenvolvimento de Algoritmos utilizando Python.

## Objetivo

Realizar os exercícios propostos utilizando Python e enviar as respostas através do GitHub.

## Fluxo de Trabalho

1. Fazer um Fork deste repositório.
2. Clonar o Fork no computador.
3. Resolver os exercícios.
4. Fazer commits periódicos.
5. Enviar um Pull Request ao final.

## Estrutura

- instrucoes → tutoriais e regras
- exemplos → exemplos de códigos em Python
- exercicios → atividades propostas
- respostas → local onde os alunos colocarão os códigos

## Regras

- Cada aluno deve trabalhar apenas no próprio Fork.
- Os exercícios devem ser feitos individualmente.
- Os commits devem possuir descrições claras.
- Todos os arquivos devem estar em Python (.py).
